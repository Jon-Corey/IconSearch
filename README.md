# Icon Search: Find Tabler Icons

Icon Search is a tool for easily finding Tabler icons.

Icon Search uses a custom search algorithm to help you quickly find exactly what you are looking for. And once you've found the icon you want, Icon Search has extensive export options to help you get exactly what you need.

Check it out at [IconSearch.App](https://iconsearch.app/)

## Comparison Details

Comparison table last updated: 20 March 2026 (Tabler version v3.40.0).

### Comparison Methodology

For each site in the list of comparable sites, I went to the site and measured it against the list of comparison criteria below.

The Comparison Criteria section below lists each criterion and its definition. Below that section is a section with further details about each site such as why it was given the score that it was for each criterion.

### Comparison Criteria

- **Up To Date** - Whether or not the site uses the latest version of Tabler Icons.
- **Export Formats** - The number of export formats that the site provides (SVG, PNG, CSS classes, data URI, etc.). Multiple formats with the same values are not counted as multiple formats. For example, the React, Vue, Preact, Solid JS, and Svelte formats are all the same and so are counted as one total format.
- **Export Customization** - How much the exported icon can be customized. This includes non-visual customizations such as adding or removing CSS classes from an exported icon. Style (outline or filled), stroke width, and color are common customizations and so are considered "Basic customization".
- **Ad-Free** - Whether or not the site includes ads. 
- **Results Sorted by Relevance** - Whether or not the search results are sorted by how relevant each icon is to the search term. For example, if a user searches for "settings", the "settings" icon should be at or near the top.
- **Search Order Flexible** - Whether or not the order of search terms can negatively affect search results. For example, if a user searches for "ringing bell", the "bell-ringing" icon should be in the search results even though the search terms are backwards.
- **Search by Symbol** - Whether or not search terms with a symbol in them return the icons that contain that symbol. For example, if a user searches for "+", all icons with a plus symbol should be in the search results.
- **Search Precision** - How precise the search is. Search results should only include results that are relevant to the search term. They should also include all results related to the search term.

### Icon Search

- **Export Formats** - 10 - SVG, Hex, HTML Char, HTML Classes, Webfont Element, React Name, React Element, Icon Name, Data URI, Base64 Data URI
- **Export Customization** - ✅ Lots of customization - Style, stroke width, color, remove extra whitespace, remove inline styles, remove xmlns attribute, add "icon" class, add Tabler classes.
- **Ad-Free** - ✅ Yes - Does provide a small Patreon icon in the site header, but it is small, out of the way, and not emphasized beyond other icons in the same area.
- **Results Sorted by Relevance** - ✅ Yes - Uses scoring system that ensures that the most relevant search results are shown first.
- **Search Order Flexible** - ✅ Yes - Searching for "ringing bell" includes the "bell-ringing" icon in the search results.
- **Search by Symbol** - ✅ Yes - Searching for "+" includes all of the icons with a plus icon in them in the search results.

## Developers

This site is a static website built with HTML, CSS, and JavaScript. It does not use any frameworks or have any build steps.

### Important Files

- `GetIconData.ps1` - A PowerShell script that downloads the latest Tabler Icons release and updates `icons.json` with the latest icons and icon data.
- `icons.json` - Contains all of the icons and icon data that is used by the site. Generated by `GetIconData.ps1`.
- `override-tags.json` - A list of additional tags to apply to specified icons. This is meant to improve search results by adding additional terms that the user can search for to find their desired icon. This is what enables users to search for "+" and find all icons with a plus symbol in them.
- `icons.js` - This is the code that handles searching, displaying, and exporting icons. This is where the bulk of the logic lives.
- `app.js` - This is the code that handles the theme (dark/light mode) and keeping the copyright year in the footer up to date.

### How to Update the Tabler Icons Version

When a new version of [Tabler Icons](https://github.com/tabler/tabler-icons) is released, this site needs to be updated to get the latest icons and icon data. This is the process to do so:

1. Run the `GetIconData.ps1` script to download the latest release and update `icons.json` with the latest icons and icon data. `GetIconData.ps1` requires [PowerShell](https://learn.microsoft.com/en-us/powershell/scripting/install/install-powershell) to be installed.
2. Check the [changelog](https://github.com/tabler/tabler-icons/releases) to ensure that there aren't any changes that would break this site (such as changes to how/where the icon data is stored).
3. Look at the name and tags of any new icons (not new styles of existing icons) and determine if there is a common search term that wouldn't result in this icon being included in the search result (such as symbols (+, -, etc.) or the expanded form of an acronym ("VIP" -> "very", "important", "person")). If that is the case, add that search term to `override-tags.json`.
4. Run this site locally and test to make sure nothing broke.
